# aiwithjen.com — Site Repository

A personal portfolio site for Jen Kelleman, built with Hugo and deployed via Netlify/Vercel.

## Local development

```bash
hugo server -D
```

## Structure

```
site/
├── content/          # Markdown pages (one per page)
├── layouts/          # HTML templates
├── static/           # Images, downloads, assets
├── assets/css/       # Stylesheets
├── config.toml       # Site configuration
└── README.md         # This file
```

## Deployment

Push to `main` → auto-deploys via Netlify/Vercel.
